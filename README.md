### WEATHER FIT TALK

날씨와 상황에 맞는 옷차림을 추천하는 로컬 LLM 기반 AI 스타일링 챗봇입니다.

사용자의 질문과 사이드바 조건을 바탕으로 날씨, 기온, 바람, 상황, 스타일을 분석하고, Ollama 로컬 LLM을 통해 현실적인 코디를 추천합니다.

OpenAI/Gemini API Key 없이 로컬 LLM으로 실행할 수 있도록 구성했으며, 필요에 따라 DuckDuckGo 검색을 함께 활용할 수 있습니다.

**Tech Stack**

`Python` · `Streamlit` · `Ollama` · `Gemma` · `DuckDuckGo` · `CSS`

**GitHub**

https://github.com/psy0635-ctrl/Weather-Fit-Talk-Local-LLM

## How to Run

초보자도 아래 순서대로 따라 하면 로컬에서 포트폴리오를 확인할 수 있습니다.

1. 레포지토리를 클론합니다.

```bash
git clone https://github.com/psy0635-ctrl/Suyong-Portfolio.git
```

2. 프로젝트 폴더로 이동합니다.

```bash
cd Suyong-Portfolio
```

3. `index.html` 파일을 브라우저에서 엽니다.

파일을 더블 클릭하거나 브라우저 창으로 드래그하면 바로 확인할 수 있습니다.

4. VS Code를 사용한다면 Live Server로 실행할 수 있습니다.

VS Code에서 `index.html`을 연 뒤, Live Server 확장 프로그램의 `Go Live` 버튼을 누르면 로컬 서버에서 확인할 수 있습니다.
